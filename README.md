# 🎮 Dashboard de Assinaturas Xbox

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um dashboard interativo no Microsoft Excel voltado para a análise de assinaturas relacionadas ao ecossistema Xbox.

O dashboard foi criado com o objetivo de transformar dados em informações visuais claras e organizadas, permitindo uma análise prática e eficiente dos serviços de assinatura.

Os serviços analisados são:

- Xbox Game Pass
- EA Play Season Pass
- Minecraft Season Pass

Os dados utilizados no projeto abrangem o período de:

📅 01/01/2026 até 31/12/2026

---

# 🎯 Objetivos do Projeto

- Organizar dados de assinaturas
- Facilitar a visualização de métricas
- Melhorar a interpretação dos dados
- Permitir análises comparativas entre serviços
- Auxiliar na tomada de decisões baseada em dados

---

# 📊 Funcionalidades do Dashboard

O dashboard permite visualizar:

- Segmentação de dados por planos anuais, mensais e trimestrais
- KPI de faturamento total do EA Play Season Pass
- KPI de faturamento total do Minecraft Season Pass
- Gráfico de faturamento do Xbox Game Pass, separado por pagamentos com e sem renovação automática
- Visualização dinâmica e interativa dos dados
